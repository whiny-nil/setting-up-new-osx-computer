Setting up VSCode
=================

* Install `code` command in terminal
  - Open the Command Palette (⇧⌘P) and type 'shell command' to find the Shell Command: Install 'code' command in PATH command.
* Change the editor font size: Preferences -> Settings -> Editor Font Size (18)
* Change the editor font
  - download and install "Fantasque Sans Mono - Normal" (TTF???)
  - Preferences -> Settings -> Editor Font Family ('Fantasque Sans Mono', Menlo, Monaco, 'Courier New', monospace)
  - Preferences -> Settings -> Font -> Enable Ligatures (set setting to true)
* Change the tab size: Preferences -> Settings -> Editor Tab Size (2)
* Change the theme: One Dark Theme by mskelton (then install bold italic sub-theme)
* Install language extensions (TS, JS, React, Vue, Ruby, whatever)
* Install ESLint
* Install Prettier
* Install Rainbow Brackets
* Install Quit Control
* Install Auto Close Tag
* Install Auto Rename Tag
* Update the terminal settings (search for settings.json in the preference search):
  "terminal.integrated.fontSize": 14,
  "terminal.integrated.fontFamily": "Monaco",
  "terminal.integrated.lineHeight": 1
